# Python-SQL based Bookstore
Python-SQL based Bookstore with functionalities to add, update, delete and search by various keywords

## Introduction
A basic application for managing books in a bookstore using Python and SQL. This application allows you to add, update, delete, and search books in a database by the title and author.
Features

* Add new books with title, author, and publication year.
* Update existing books by changing their title, author, and publication year.
* Delete books from the database.
* Search for books by title, author, or publication year.
